Featured Project • AI / ML / Security

AI-Powered Vulnerability Assessment Assistant

Built an ML-based tool that integrates with Nessus and OpenVAS APIs to prioritize vulnerabilities more intelligently. Instead of treating every finding the same, the system uses contextual risk scoring based on exploitability, exposure, and asset value.

  • Processed 1000+ findings through an automated prioritization pipeline.
  • Improved prioritization efficiency by 40%.
  • Applied attack-path analysis to identify high-risk lateral movement scenarios.
  • Automated stakeholder-specific reports, reducing manual effort by 50%.
  • Containerized deployment with Docker for portability and scalability.
Data Analytics / Machine Learning

HR Analytics — Attrition and Salary Loss

Developed models to predict employee attrition and forecast salary growth, with the goal of helping organizations identify retainable employees and quantify hidden salary loss more effectively.

  • Achieved AUC-ROC of 0.93 for attrition prediction using Decision Tree.
  • Best regression model was Random Forest with R² = 0.44.
  • Identified 733 retainable employees — around 50% of the dataset.
  • Estimated $12.1M in annual salary loss.
  • Evaluated results with F1-score, precision, recall, AUC-ROC, RMSE, and MAPE.
